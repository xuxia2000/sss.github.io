文件上传

前端代码

servlet 实现

servlet文件上传中最重要的类 part

part的获取

Part part = request.getPart(“MyFile”);

part类获取文件名

String filename = part.getSubmittedFileName();

part类获取输入流

InputStream is = part.getInputStream();

获取文件存放路径

String realPath = request.getServletContext().getRealPath(“/“);
File file = new File(realPath+filename);
也可以自己决定存放路径
File file = new File(“F:\myblogssm\“ + filename);

获取输出流

FileOutputStream fos = new FileOutputStream(file);

通过输出流写入服务器文件系统

byte[] bytes = new byte[1024];
int length = 0;
while ((length = is.read(bytes))!= -1){
fos.write(bytes,0,length);
}

part转存文件
part.write() // 参数是string,直接放存入的路径

springboot实现

其中最重要的类 MultipartFile
获取MultipartFile类

直接通过参数传递
public String onfile(MultipartFile MyFile) throws IOException

MultipartFile获取文件名

String filename = MyFile.getOriginalFilename()

MultipartFile获取输入流

InputStream inputStream = MyFile.getInputStream();

获取文件存放路径

直接自己设置
File file1 = new File(“F:\myblogssm\“ + MyFile.getOriginalFilename());

获取输出流

FileOutputStream fos = new FileOutputStream(file1);

写入文件系统

byte[] bytes = new byte[1024];
int length = 0;
while ((length = is.read(bytes))!= -1){
fos.write(bytes,0,length);
}

直接转存文件
MyFile.transferTo(file1); //可以接收文件类，也可以接收路径字符串

文件下载

前端代码

servlet实现

获取文件名，该参数由前端传递而来

String file = request.getParameter(“file”);

获取文件存储路径

String path = request.getServletContext().getRealPath(“”)+”/“+file; //默认认为文件在当前项目的根目录

根据路径获取输入流（先将文件输入到内存）

FileInputStream fis = new FileInputStream(path);

设置请求头信息

response.setCharacterEncoding(“utf-8”);
response.setHeader(“Content-Disposition”, “attachment; filename=”+file);

获取输出流

ServletOutputStream out = response.getOutputStream();

将内存里的文件通过输出流返回

byte[] bt = new byte[1024];
int length = 0;
while((length=fis.read(bt))!=-1){
out.write(bt,0,length);
}
out.close();
}

springboot实现

获取文件名（通过@PathVariable可实现restful风格）

@GetMapping(“download/{filename}”)
public ResponseEntity<byte[]> download(@PathVariable String filename) throws IOException {

获取文件的存储路径

String filepath= “F:\myblogssm\“+filename;（根据自己上传路径写）

根据路径获取输入流（先将文件输入到内存）

File file =new File(filepath);
//创建字节输入流，这里不使用Buffer类
InputStream in = new FileInputStream(file);

将服务器存储的文件读入到内存

//available:获取输入流所读取的文件的最大字节数
byte[] body = new byte[in.available()];
//把字节读取到数组中
in.read(body);

设置请求头

MultiValueMap<String, String> headers = new HttpHeaders();
headers.add(“Content-Disposition”, “attchement;filename=” + file.getName());
//设置响应状态
HttpStatus statusCode = HttpStatus.OK;

将存储了信息的数组封装到ResponseEntity类中

ResponseEntity<byte[]> entity = new ResponseEntity<byte[]>(body, headers, statusCode);
return entity;//返回

springboot完整下载代码

总结

上传时通过 par t或 MultipartFile 类获取输入流，通过文件路径获取输出流，将输入流中的信息通过输出流写入文件系统，下载时通过文件路径获取输入流，通过response.getOutputStream 然后将信息全部写入输出流中，springboot是直接封装了输入流存储信息的字节数组。

除此之外，要记得设置请求头的状态
